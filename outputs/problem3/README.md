# Problem 3 Output Package

This folder is the formal Problem 3 dynamic-response scenario package for the
current modeling round.

Problem 3 has no official dynamic event stream in the statement. The four
scenario folders here are therefore representative scenario assumptions, not
hidden official data. The assumptions are recorded explicitly in
`scenario_assumptions.csv`.

## Baseline

The dynamic response starts from the formal Problem 2 `DEFAULT_SPLIT` result:

- Baseline folder: `outputs/problem2/default_split/`
- Baseline total cost: `49239.78`
- Policy conflicts: `0`
- Physical vehicles: `E1:10, F1:35`
- Late stops / maximum lateness: `12 / 129.44 min`

The mainline inherits the Problem 2 green-zone policy: fuel vehicles may not
serve green-zone customers during `[480,960)`.

## Reproduction

The bounded data refresh command is:

```powershell
python problems/problem3.py --iterations 8 --remove-count 4 --seed 20260426 --output-dir outputs/problem3 --no-plots
```

PNG plots were generated by an earlier complete output pass. For future
refreshes, prefer bounded per-scenario commands or `--no-plots`; a previous
all-scenario plot rerun timed out and was documented in `.learnings/ERRORS.md`.
